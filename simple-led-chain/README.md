# 01 - Simple Led Setup

## Date
May 22, 2026

## Goal
Build a basic Arduino setup that displays leds in response to user button input

## What I Built
I followed the first lesson from the Arduino kit book and got the circuit/program working.

## Parts Used
- Arduino Uno R4
- Breadboard
- Jumper wires
- Button
- 2 red 1 green diode LEDS
- 222, 10k ohm resistors

## What I Learned
- the `setup()` function runs once and only once when the Arduino starts.
- `loop()` is a function that repeats continuously through runtime.
- The digital pins can be set to either high or low
